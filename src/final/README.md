# Camada FINAL (Gold)

A camada FINAL contém o modelo dimensional utilizado para consumo análitico, ou seja, tabelas classificadas como fato e dimensão.
Nessa camada são criadas views, tabelas finais, métricas, agregações e estruturas otimizadas para BI.
É a camada utilizada para alimentar dashboards, análises e ferramentas como Power BI e Tableau.

OBJETIVO: Disponibilizar dados modelados e prontos para uso de negócio.
