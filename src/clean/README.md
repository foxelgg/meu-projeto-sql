# Camada CLEAN (Silver 2)

Essa camada recebe os dados já corrigidos e padronizados oriundos da camada STAGING.
Nesta camada, os dados são atribuídos aos seus tipos corretos, bem como são atribuídas também Primary Keys e Foreign Keys, constraints e integridade referencial.
A camada CLEAN apresenta dados confiáveis e prontos para uso analítico.

OBJETIVO: Entregar dados sólidos, confiáveis, definitivos e prontos para análise.
